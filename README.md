Indian Stock Dashboard is a modern, interactive web application for tracking and analyzing Indian stock market data. Built with FastAPI, Python, and vanilla HTML/CSS/JavaScript, this app allows users to monitor their favorite stocks, view historical trends, set price alerts, and manage a personalized watchlist.

Features 

Real-time Stock Data: Fetches live stock prices and historical trends for Indian companies.

Interactive Charts: View zig-zag style stock charts for multiple timeframes: 1M, 3M, 6M, 1Y, 3Y, 5Y, 10Y, MAX.

Watchlist: Add/remove stocks to a personal watchlist with one click.

Price Alerts: Set alerts for when a stock crosses a target price (above or below).

Persistent State: Last searched stock and timeframe are remembered between sessions.

Dark Theme: Eye-friendly dark mode for comfortable viewing.

Responsive UI: Works on desktop and mobile devices.

User Authentication: Secure login/signup with JWT tokens and password hashing.

Tech Stack 

Backend: Python, FastAPI, SQLAlchemy, PostgreSQL

Frontend: HTML, CSS, JavaScript, Chart.js

Authentication: JWT, Argon2 for password hashing
